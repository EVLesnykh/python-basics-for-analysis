# Python Data Analysis Exercises

Этот репозиторий содержит мои учебные и практические задания по анализу данных на Python.

## Используемые инструменты
- Python
- pandas
- numpy
- matplotlib / seaborn

## Темы
- загрузка и очистка данных
- EDA (exploratory data analysis)
- визуализация
- работа с пропусками

Цель — закрепление навыков анализа данных на реальных и учебных датасетах.
